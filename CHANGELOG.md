## Unreleased

### Added
- Monorepo scaffolding (pnpm workspaces + Turborepo).
- `@load-games/core` engine primitives: `BaseEngine`, `GameLoop`, `InputManager`.
- `@load-games/react` `<GameCanvas/>` wrapper.
- Six game engines: snake, flappy, breakout, pong, runner, space-invaders.
- Demo app (`apps/demo`) — Vite + React playground with tabs, theme/speed controls, callback log.
- Changesets workflow for per-package versioning.
- GitHub Actions CI: build + typecheck + test on Node 20 & 22.
- GitHub Actions release workflow (Changesets → npm publish).
- GitHub Pages deploy for demo.

### Fixed
- Root `packageManager` field added so Turbo can resolve workspaces.
- Demo App typed against `BaseEngine` not anonymous `{start,destroy}`.
- Package `exports` map ordered `types` → `import` → `require` (resolves tsup warning).
- `@load-games/core` test script uses `--passWithNoTests` until core tests land.
